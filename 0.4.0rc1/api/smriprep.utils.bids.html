smriprep.utils.bids.predict_derivatives(subject_id, output_spaces, freesurfer)[source]¶

Generate a list of the files that should be found in the output folder.

The prediction of outputs serves two purposes:

  • Anticipates the set of outputs that will be generated.

  • Informs the decision of whether the workflow should be staged or some sections can be skip.

Parameters

  • subject_id (str) – A subject id

  • output_spaces (list) – TemplateFlow identifiers of the requested output spaces

  • freesurfer (bool) – Whether the --fs-no-reconall flag was used.

Examples

>>> predict_derivatives('01', ['MNI152NLin2009cAsym'], False)
